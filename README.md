# Discount Service

Сервис расчёта скидок.

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/health` | Проверка здоровья |
| POST | `/discounts/calculate` | Рассчитать скидку |

## Правила скидок

| Условие | Скидка |
|---------|--------|
| Промокод `STUDENT10` | 10% |
| Количество ≥ 10 | 5% |
| Промокод `WELCOME5` | 5% |
| Количество ≥ 5 | 3% |

## Запуск

```bash
make run
```
